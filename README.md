# hytrip

橙白旅途，租房用概念项目，技术栈包括：vue3, pinia, vue-router, vant, axios, dayjs等

应用了**路由分包**，**懒加载**，第三方插件按需引入等性能优化技术

## 启动项目

```sh
npm install
npm run dev
```

## 打包

```sh
npm run build
```
